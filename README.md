# 中华美食天地

这是一个展示中华美食文化的静态网站项目。本项目使用纯前端技术（HTML5、CSS3、JavaScript）构建，旨在提供优雅、直观的美食信息浏览体验。

## 🚀 项目预览

[点击这里查看在线演示](https://ZELO-EX.github.io/Chinese-Foods/index.html)

## 📖 项目介绍

该项目是一个内容丰富的中华美食门户网站，包含以下核心功能：
*   **用户系统**：用户注册、登录、密码重置。
*   **美食浏览**：首页概览、菜系介绍、经典菜品展示、详细菜谱查阅。
*   **响应式设计**：专为PC端适配，布局合理，视觉层次清晰。

## 🛠️ 技术栈

*   **前端三件套**: HTML5 + CSS3 + JavaScript
*   **布局技术**: DIV+CSS、Flexbox/Grid 响应式布局
*   **数据交互**: `fetch` API 进行同源请求，使用 `async/await` 处理异步操作
*   **本地存储**: 使用 `cookie` 和 `localStorage` 管理用户登录状态与数据
*   **交互功能**: 轮播图、下拉菜单、表单验证、页面平滑跳转、动态内容加载

## 🎨 设计特点

*   **视觉风格**: 采用具有中国传统文化韵味的深褐色(`#3c2f2f`)、金色(`#d4af37`)和米白色(`#f8f5f0`)作为主色调，搭配中国红(`#c00`)作为点缀。
*   **字体选择**: 标题使用书法风格字体，正文使用清晰易读的字体，提升阅读体验。
*   **内容呈现**: 高质量美食图片配以简洁明了的文字描述，图文比例协调。
*   **交互体验**: 导航栏悬停效果、轮播图（支持自动切换与手动控制）、图片悬停放大等细腻的交互反馈。

## 💡 难点与总结

在项目开发过程中，主要挑战和收获包括：
*   **布局适配**: 针对不同内容模块选择合适的布局方案（Flexbox/Grid），以创造灵动且和谐的视觉体验。
*   **代码抽象**: 对菜系介绍、菜谱详情等具有重复结构的页面进行抽象化设计，有效减少了代码冗余，提高了可维护性。
*   **细节打磨**: 在轮播图交互、表单验证、动画效果等细节上投入了大量精力，这要求掌握远超基础知识的前端技能。
